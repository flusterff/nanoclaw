## 생성 프롬프트 주입 규약
- **씬 생성/리라이트 프롬프트**: 절대 규칙 1·6a-6e·7·8·9·10 인라인 + 이 원장의 [scene] 행 요약.
- **contract/대본 생성 프롬프트**: 규칙 2·3·4 + 리뷰어 A 체크리스트 + [contract] 행 요약.
- **래스터 컷 생성**: 규칙 11 프로토콜.
- **런타임(V2/V3) 설계 시**: 이 원장을 엔진측 fixtures/가드 후보 목록으로 사용.

## 검증 루프
생성 직후: `scripts/scene-regression-scan.py <씬.html>` (기계) → 리뷰어 B(vision, 항목 1-8) → Will 눈확인(이상치·샘플). 스캔 FAIL = 리뷰어 도달 전 자동 반려.
