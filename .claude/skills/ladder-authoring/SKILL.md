# /ladder-authoring — 탐구형 레슨 저작 하네스 v2

**성립 배경 (2026-07-02, Will 품질 개입):** ① SVG로 그린 사람 손·목 = 퀄리티 최악 ② "밀대"(스프링+눈금 가상 기구) = 초안 모델이 발명하고 판정 모델이 자기채점 통과 → 어른도 모르는 장치가 레슨에 들어감. 원인 = 게이트("만9 친숙도")가 honor-system 자기채점. 해법 = 외부 앵커(화이트리스트) + 독립 판정(아이눈 리뷰어) + 자동 반려.
